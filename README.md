# GDOCS Clone

This is a minimalist clone of the Google Docs service. Once development is complete, more than one person can collaboratively work on a document together.

## Tech stack

1. [Node.js](https://nodejs.org/)
2. [React.js](https://reactjs.org/)
3. [Firebase](https://firebase.google.com/)
4. [Google Cloud Platform](https://cloud.google.com/)
5. [MaterialUI](https://material-ui.com/)
